# Мои команды для экзамена по сетевым технологиям

Содержит полные настройки для ISP, HQ-RTR, BR-RTR, HQ-SRV, BR-SRV.

## Как найти этот файл?
Введите в поиске любую из фраз:
* `экзамен сети isp hq-rtr`
* `настройка ospf frr vtysh`
* `dnsmasq au-team.irpo`

## Содержание
* Команды для ISP (NAT, iptables)
* Команды для HQ-RTR и BR-RTR (OSPF, GRE, DHCP)
* Команды для HQ-SRV (dnsmasq, SSH)
